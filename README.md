# Movie Recommendation & Analysis System

## Overview
This project implements a complete **machine learning pipeline** for analyzing movie data and building a **content-based recommendation system** along with a **supervised regression model** for rating prediction.

The focus of the project is on:
- Clean project structure
- Separation of logic and experimentation
- Correct use of supervised and unsupervised learning concepts

 ## Dataset
 Kaggle database- https://www.kaggle.com/datasets/anandshaw2001/movie-rating-dataset
